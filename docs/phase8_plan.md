# Phase 8 Plan - Proposal Governance (GovDash Parity)

## Goals
- Deliver proposal governance features that mirror GovDash Proposal Cloud.
- Strengthen compliance controls, review workflows, and outline governance.

## Deliverables
1. Compliance Shreds + Document Classification
   - Parse RFP docs into shreds (tagged fragments).
   - Tag shreds by section, requirement type, and status.
   - Provide shred filters and grouping in UI.

2. Proposal Outline Editor
   - Volume/section hierarchy with drag-and-drop ordering.
   - Page limits and reviewer assignments per volume/section.
   - Outline-to-section mapping with compliance requirements.

3. Review Mode + Redlines
   - Reviewers can comment, request changes, and mark approvals.
   - Redline tracking against generated and edited content.

4. Compliance Package Export
   - Export compliance matrix + outline + requirement mapping in one bundle.

5. Labor Categories + Key Personnel
   - Match labor categories to key personnel and past performance.
   - Adjudication workflow for competing candidates.
