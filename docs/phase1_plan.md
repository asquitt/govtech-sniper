# Phase 1 Plan - Discover + Capture MVP

## Goals
- Strengthen the Discover workflow (opportunity listing, scoring, and filters).
- Introduce a capture pipeline with bid decisions, gate reviews, and teaming partners.
- Ensure audit logging and webhook events are emitted for capture actions.

## Scope
- Capture Plan model + CRUD endpoints.
- Gate Review model + CRUD endpoints.
- Teaming Partner directory + link to opportunities.
- API endpoints + tests for all capture features.

## Deliverables
1. Capture pipeline endpoints under `/api/v1/capture`.
2. Teaming partners linked to opportunities.
3. Audit and webhook coverage for capture actions.
4. Tests covering plan creation, updates, gate reviews, and partner linking.

## Status
- Implemented capture plan, gate reviews, teaming partners, and tests.
- Added discover scoring improvements, opportunity detail view, and snapshot diff UI.
- Phase 1 complete.
