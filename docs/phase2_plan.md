# Phase 2 Plan - Proposal Cloud MVP

## Goals
- Expand proposal generation into a full Proposal Cloud with compliance management and submission packaging.

## Deliverables
1. Compliance Matrix editor UI improvements (tagging, assignment, status).
2. Proposal project workspace with sections, approvals, and versioning.
3. Submission package tracking + export enhancements.
4. Tests for proposal CRUD, section generation, and export workflows.

## Work Breakdown
- Backend:
  - Proposal project model extension (versions, submissions).
  - Endpoints for proposal lifecycle management.
  - Export improvements (DOCX/PDF with metadata).
- Frontend:
  - Proposal workspace UI.
  - Section editor with citation panel.
  - Submission tracker UI.

## Status
- In progress.
- Compliance matrix editor endpoints + UI editor added (tags, assignment, status).
- Proposal workspace with section editing, evidence linking, and submission packages added.
- Version history UI for proposals added.
